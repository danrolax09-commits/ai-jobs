# AIJobs Board

The leading job board for AI engineers and AI companies.

## Features

- **50K+ AI engineers** - Access the largest community of AI professionals
- **2,800+ active jobs** - Roles from startups to Fortune 500 companies
- **Job posting** - Post for just $99, reach qualified candidates instantly
- **Analytics** - Track applications and hiring metrics
- **Email digest** - Featured jobs sent to subscribers
- **Salary transparency** - All salaries listed upfront

## Revenue Model

- **Job posting:** $99 per job listing
- **Featured listing:** $199 (top 3 positions)
- **Sponsored jobs:** $499 (featured + newsletter highlight)
- **Recruiter pro:** $299/month (unlimited postings)

Expected monthly revenue: $5-8K (50-80 job postings/month).

## Quick Start

```bash
npm install
npm run dev
```
